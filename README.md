## Opis projektu

Projekt przedstawia stronę CV wykonaną w HTML oraz ostylowaną przy użyciu zewnętrznego arkusza CSS.

Celem zadania było oddzielenie struktury dokumentu (HTML) od warstwy prezentacji (CSS).

## Struktura projektu

W folderze `zad2_css` znajdują się:

- `index.html` – struktura strony
- `styles.css` – arkusz stylów

## Zastosowane rozwiązania

W projekcie zastosowano:

- zewnętrzny arkusz stylów (`link rel="stylesheet"`)
- kolor przewodni w odcieniu zieleni
- zmianę kroju pisma (font-family)
- marginesy i odstępy (margin, padding)
- stylowanie nagłówków, sekcji oraz linków

Strona posiada wyraźne formatowanie wizualne i różni się od czystego dokumentu HTML.
